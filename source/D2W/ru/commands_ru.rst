D2W: Команды
~~~~~~~

.. Important::
   Возможность управления по локальной сети без токена безопасности по умолчанию `заблокирована <safety_ru.html>`_ в целях безопасности. Это может быть актуальным, например, при использовании локального управления в общественных местах. Если данная мера безопасности не нужна, отключите ее на устройстве, переведя параметр ``bLc`` в значение **oFF**. В противном случае запросы на изменение без данного токена не будут выполняться.

Для получения параметров устройства служит ключ ``cmd``. 
Возможные значения:

   * 1 - запрос на получение `параметров <parameters_ru.html>`_
   * 3 - запрос на получение времени устройства
   * 4 - запрос на получение `телеметрии <telemetry_ru.html>`_

Запрос ``{"cmd":1}``

Получение `параметров <parameters_ru.html>`_

Ответ: 

.. code-block:: json

   {
      "deviceId":0,"par":{"1":245,"5":144,"9":3,
      "14":-8,"20":3,"19":3,"13":1,"73":5,
      "124":false,"125":true,"126":false,"119":true,
      "118":true,"117":true,"116":false,"114":false,
      "115":false,"ts":1781076688}
   }

Запрос ``{"cmd":3}``

Получение времени устройства:

Ответ: 

.. code-block:: json

   {"deviceId":0,"time":834401814}

Где ``time`` - время в секундах от 01.01.2000 00:00 в UTC  

Запрос ``{"cmd":4}``

Получение `телеметрии <telemetry_ru.html>`_

Ответ:

.. code-block:: json

   {
     "deviceId":0,"telemetry":{"u":{"13":245,"14":144,"15":218},
     "t":{"0":576,"7":946},"m":{"1":0,"3":0,"10":0},
     "f":{"0":false,"1":false,"2":false,"3":false,"4":true,"5":true,
     "6":false,"10":true,"12":false,"13":false,"14":false,"15":false},
     "o":{"0":-58},"ts":834392219}
   }
