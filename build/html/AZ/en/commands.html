commands¶

To get device parameters use cmd key. Possible values:

  • 1 - parameters request

  • 2 - schedule request

  • 3 - time getting request

  • 4 - telemetry request

  • 5 - parameters reset request

  • 6 - schedule reset request

  • 10 - schedule request for devices with air sensor
